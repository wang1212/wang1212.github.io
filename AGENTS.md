# AGENTS.md

This is a personal blog project built with Docusaurus, named "不如怀念".

## Blog Structure

Blog posts are located in the `blog/` directory with the following categories:
- `awesome/`: Featured technical articles
- `business/`: Business-related content
- `computer-technology/`: Technology articles
- `finance-economics/`: Finance and economics content
- `life/`: Life records and experiences
- `tools/`: Tools and tutorials
- `weekly/`: Weekly reviews

## Blog Post Front Matter

Each blog post contains front matter with the following fields:
- `title`: The title of the post
- `date`: The creation date in YYYY-MM-DD HH:mm:ss format
- `update`: The last update date in YYYY-MM-DD HH:mm:ss format (must be updated when content changes)
- `authors`: Author information (references authors.yml)
- `tags`: Array of tags for categorization
- `keywords`: Reference to tags
- `description`: Brief description of the post

**Important**: When updating blog post content, the `update` field must be updated to reflect the modification time.
